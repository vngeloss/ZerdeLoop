


ZerdeLoop- это бесплатная платформа обмена знаниями для сельских школьников Казахстана, готовящихся к ЕНТ. Проект решает проблему нехватки репетиторов в регионах без использования денег.

Коротко о принципе работы нашего проекта:
Пользователи заполняют анкету, указывая предмет для изучения и предмет, который могут объяснить сами. Алгоритм анализирует базу данных в реальном времени и объединяет участников в замкнутые цепочки взаимопомощи из двух или трех человек. Например, первый участник учит второго, второй учит третьего, а третий учит первого.

Технический стек :
Интерфейс: HTML, CSS (Tailwind CSS), шрифты Nunito и Manrope.
База данных: Supabase (PostgreSQL).
Алгоритм: Скрипт на JavaScript для автоматического поиска циклов в графе.
